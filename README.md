# أدب كاست (AdabCast)

منصة أدبية عربية معاصرة للاستماع إلى القصائد والنصوص الأدبية.

## الميزات

- **مشغّل صوتي متقدم**: مع قائمة تشغيل، شريط تقدم، وتحكمات كاملة
- **واجهة مستخدم عصرية**: تصميم داكن مع دعم الوضع المظلم
- **تصميم متجاوب**: يعمل على جميع الأجهزة
- **دعم اللغة العربية**: خطوط وتخطيط مناسب للنصوص العربية
- **مكتبة محتوى**: مجموعة متنوعة من القصائد والنصوص
- **استوديو إنشاء**: للمبدعين لرفع أعمالهم
- **ملف شخصي**: لإدارة المحتوى والإعدادات

## التقنيات المستخدمة

- **HTML5**: هيكل الصفحات وSemantic HTML
- **Tailwind CSS**: إطار عمل CSS للتصميم
- **JavaScript**: تفاعل المستخدم والمشغّل الصوتي
- **Google Fonts**: خطوط Noto Sans Arabic و Amiri
- **Material Symbols**: أيقونات Google

## كيفية التشغيل

1. قم بتحميل جميع الملفات إلى مجلد واحد
2. افتح `index.html` في متصفح الويب
3. أو شغّل خادم محلي:

```bash
# باستخدام Python 3
python -m http.server 8000

# أو باستخدام Node.js
npx serve .

# ثم افتح http://localhost:8000
```

## الصفحات

- `index.html`: الصفحة الرئيسية
- `player.html`: مشغّل الصوت
- `library.html`: مكتبة المحتوى
- `texts.html`: معرض النصوص
- `submit.html`: نموذج التقديم
- `studio.html`: استوديو الإنشاء
- `profile.html`: الملف الشخصي

## الملفات

- `styles.css`: جميع أنماط CSS
- `script.js`: JavaScript للتفاعل
- `tailwind.config.js`: إعدادات Tailwind (مدمجة في script.js)

## الميزات الإضافية

- **الوضع المظلم**: تبديل تلقائي مع حفظ التفضيل
- **اختصارات لوحة المفاتيح**: Space للتشغيل، الأسهم للتنقل
- **زر العودة للأعلى**: يظهر عند التمرير
- **تحميل مسبق**: شاشة تحميل أنيقة
- **إمكانية الوصول**: دعم قارئات الشاشة واختصارات التنقل

## التطوير

للتطوير والتعديل:

1. قم بتعديل الملفات المطلوبة
2. اختبر التغييرات محلياً
3. تأكد من عدم وجود أخطاء في وحدة التحكم
